# Reports Microservice
Description: Report microservice generates any of the following reports
1. Category Count Report - provides count in specific category requested
2. Statistics Report - provides numerical stats such as total, mean, median, etc
3. Filter by Date Report - provides specified columns for time range

# How to REQUEST data from reports microservice
To request data from microservice, send a http post request with JSON data

For Category Count Report:  
- requests.post("https://cs361-reports-microservice-production.up.railway.app/report/count", json={"application_data": test_data, "category": category})

# How to RECEIVE data from reports microservice
To receive data from reports microservice, store json response generated by microservice

- Example filter by JSON date report:
response = requests.post("https://cs361-reports-microservice-production.up.railway.app/report/filter_date", json={"application_data": test_data, "date_column": date_column, 
        "dates": {"start_date": start_date, "end_date": end_date}})
print(response)
